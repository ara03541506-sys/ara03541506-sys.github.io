はじめに

今回はタイトルの通り、Macで仮想環境を構築し、Pythonのコードを動かす方法を解説します。 なぜわざわざ文章で書くかというと、WindowsとMacではコマンドが微妙に異なるからですね。

フォルダを作成

まずは作業用のフォルダを作成します。私は以下のような構成にしています。 

workspace/
 ├── venv/        ← 仮想環境を置く場所
 └── classes/     ← Pythonファイルを置く場所

VSコードを起動し、ターミナルから作業します。 以降のコマンドはすべてターミナルに入力します。

仮想環境を構築

まずはPythonの仮想環境を作成します。 Macの場合、以下のコマンドを入力します。 

python3 -m venv venv

Windowsでは python -m venv venv と書いていましたが、Macでは python3 になります。 理由は軽く読みましたが理解する必要はなさそうなので、そういうものだと思いましょう。

仮想環境を有効化

次に、作成した仮想環境を有効化します。 Windowsとコマンドが異なります。

  • Windows：.\venv\Scripts\activate
  • Mac：source venv/bin/activate

Macでは「source」というコマンドを使って仮想環境を起動します。 また、Windowsでは Scripts フォルダ内にactivate.batがありますが、 Macでは bin フォルダの中にあります。

コマンド実行後、ターミナルの先頭に (venv) と表示されれば成功です。

Pythonコードを実行

仮想環境を有効化できたら、次にPythonファイルを動かしてみましょう。

まず classes フォルダに hello.py というファイルを作成します。 中身は以下の1行だけです。 

print("Hello, World!")

実行は以下のコマンドで行います。 

python3 -m classes.hello

ターミナルに「Hello, World!」と表示されたら成功です。 これでMac上にPythonの仮想環境を構築し、コードを実行できるようになりました。
